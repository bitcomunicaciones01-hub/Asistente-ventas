# 🧞 Sales Assistant AI (FastAPI + WooCommerce + Instagram)

Este es un asistente de ventas de elite diseñado con un diseño **Premium Glassmorphism** y un cerebro potenciado por **GPT-4o-mini**. Es capaz de consultar stock en tiempo real en WooCommerce y automatizar la prospección en Instagram.

## 🚀 Estado Actual del Proyecto
- **Frontend**: Widget de chat con grilla horizontal de 3 productos, diseño modernista e interactividad con `marked.js`.
- **Backend**: FastAPI con monitoreo de Instagram en segundo plano.
- **Integraciones**: WooCommerce (Stock) e Instagram (DMs).

## 🛠️ Cómo seguir mañana
1.  **Clonar**: `git clone <tu-repo>`
2.  **Configurar**: Crear un archivo `.env` con las claves (OpenAI, Woo, IG).
3.  **Lanzar**: Correr `uvicorn main:app` o subir a Railway.
